# 세레브로 로딩 — 시작 인물(히어로) 아트 에셋

로딩 인트로의 **클로즈업 시작 인물**에 쓰이는 고품질 이미지/일러스트를 여기에 둡니다.

## 사용법
- 이 폴더(`apps/web/src/assets/hero/`)에 이미지 파일 **하나**를 넣으세요.
  허용 확장자: `.png` `.jpg` `.jpeg` `.webp` `.avif`
- 빌드/개발 서버가 자동으로 감지해 히어로로 사용합니다(코드 수정 불필요).
- 파일이 없으면 코드가 생성한 절차적 플레이스홀더가 대신 표시됩니다.
- 파일이 여러 개면 그중 하나가 선택됩니다(하나만 두는 걸 권장).

## 권장 사양
- **세로 인물 구도**(상반신 또는 전신). 권장 비율 ~2:3 (예: 1024×1536).
- **배경 투명(PNG/WebP 알파)** 권장 — 인물만 떠 있고 군중/배경은 3D 씬이 채웁니다.
  알파가 없으면 사각 이미지가 그대로 보이니, 어두운 단색/그라데이션 배경이 무난합니다.
- 인물이 프레임 중앙, 머리가 위쪽 1/3 지점에 오도록.
- 풀컬러로 제공하세요. 코드가 **풀컬러 → 회색 탈색 → 백색 버스트 → 디졸브**를 연출합니다.

## ⚠️ 개인정보(PIPA)
- **특정 실인물을 식별 가능한 사진/초상은 사용 금지.** 일반(익명)·가상의 인물,
  보유 권리가 있는 일러스트, 또는 라이선스가 명확한 이미지만 사용하세요.
- 출처/라이선스는 PR 또는 ADR에 기록합니다.
