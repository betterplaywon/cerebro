# ADR-0003 — 구글 소스는 "엄선 도메인"으로 (무료 전체웹 검색 종료)

- 상태: Accepted
- 일자: 2026-06-26

## 맥락
2026-01-20 구글이 Programmable Search Engine / Custom Search JSON API의 **무료 "전체 웹 검색"을 종료**했다.
- **신규** 검색엔진(2026-01-20 이후 생성)은 무료 티어에서 **최대 50개 도메인**만 검색 가능.
- **기존**에 "전체 웹"으로 설정된 엔진만 **2027-01-01까지** 유지.
- 전체 웹 검색은 유료(Vertex AI Search / 별도 등록)로만 가능.

cerebro가 새로 만든 CSE는 무료 전체웹 검색이 불가하다. 따라서 구글을 "웹 전체 보조 소스"로 쓰려던 초기 가정이 무너졌다.

## 결정
구글 어댑터를 **"엄선 도메인(≤50) 보조 소스"** 로 운용한다.
- CSE에 네이버가 약한 고가치 한국 도메인(나무위키·지식백과·스타트업/기업 DB·전문 경제/IT 매체·앱스토어)을 등록.
- **어댑터 코드는 변경 없음** — CSE가 검색하는 범위를 그대로 받아온다(엄선 도메인 결과).
- 키 미설정 시 자동 비활성(`isEnabled()=false`). 무료 MVP의 1차 커버리지는 **네이버(넓은 한국어)+위키백과**가 담당.
- 전체웹/유료(Vertex AI Search)는 추후 BM 확장 시 재검토.

## 대안
- **구글 완전 제외**: 운영 단순하나 일부 고가치 도메인(나무위키 등) 커버 손실 → 보조로 유지가 이득.
- **유료 전체웹**: 무료 MVP 원칙 위반 → 보류.
- **타 검색 API(Algolia/자체 인덱스 등)**: 향후 조사 대상.

## 트레이드오프
엄선 도메인은 전체웹보다 커버리지가 좁다. 대신 무료·합법(구글 인덱스/스니펫 사용)·네이버 보완이라는 이점이 크다. CSE 도메인 목록 유지(운영 비용)가 생긴다.
