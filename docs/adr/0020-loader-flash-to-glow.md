# ADR-0020 — 로딩 백색 버스트를 부드러운 글로우로 전환(접근성)

- 상태: Accepted (오너 요청 — "로딩 섬광이 웹 접근성·UX에 부정적, 없애거나 다른 방식으로")
- 일자: 2026-06-28
- 대상: 시네마틱 로더(세레브로 연출)의 `flash`/`bgWhiteT` 연출. 카메라 연출·군중·탈색·디졸브 등 나머지 타임라인은 불변.

## 맥락
세레브로 로딩(X-Men 오마주)은 인트로 ~3.3초에 **백색 버스트**를 터뜨린다: 배경/포그가 네이비→흰색으로 보간(`bgWhiteT`가 1.0까지), Bloom 0.7→1.6, 히어로·군중 밝기 급상승. 곡선 자체는 `bell()` 코사인이라 빠른 스트로브는 아니지만, **전체 화면 휘도가 크게 출렁이는 일반 플래시(general flash)** 다. 게다가 루프(9초 주기)마다 `miniFlash`가 반복돼, 로딩이 길어질수록 같은 번쩍임이 되풀이됐다.

광과민성·UX 관점에서 부정적이라는 오너 지적은 타당하다. 특히 **반복성**(루프 미니플래시)이 문제.

## 결정
번쩍임을 **타임라인 소스에서** 줄여 은은한 글로우로 바꾼다. `flash`·`bgWhiteT`는 세 소비처(SceneGrader의 배경/포그/Bloom, HeroPortrait의 백색 리프트, CrowdField의 `uFlash`)가 공유하는 단일 진실원이라, 소스만 고치면 전부 일관되게 부드러워진다.

- `intro.burstPeak` 0.85 → **0.28**(글로우 세기 상한), `intro.burstHalf` 0.3 → **0.6**(더 느리고 완만한 곡선).
- `bgWhiteT`에 상한 도입: `intro.bgWhitePeak = 0.12`를 곱해 배경 화이트닝을 1.0 → **0.12**로 캡(전체화면 백색 플래시 방지). `bgWhiteHalf` 0.55 → 0.9.
- `bloom.intensityBurst` 1.6 → **0.95**(Bloom 버스트 절제).
- 루프 `miniFlash*` 상수·계산 **완전 제거** → 루프의 `flash`·`bgWhiteT`는 항상 0(반복 번쩍임 소멸). 카메라 주목/후퇴/유영 연출은 유지.
- HeroPortrait의 `uBurst = flash / burstPeak`(정규화 → 항상 85% 백색) → `uBurst = flash`(글로우 세기에 직접 비례). 배경이 거의 안 밝아지는데 히어로만 새하얗게 튀는 부조화 방지.

`prefers-reduced-motion`은 별도 처리 불필요 — 해당 사용자는 시네마틱 씬 자체가 마운트되지 않고 정적 셸로 폴백(`CerebroLoader`)되어 `three`조차 받지 않으므로 섬광을 애초에 보지 않는다.

## 대안
- **섬광 완전 제거(`flash`/`bgWhiteT` 전부 0)**: 가장 안전하나 세레브로 오마주의 시그니처 비트가 사라진다. 시네마틱 정체성 유지를 위해 기각, 절제된 글로우로 절충.
- **`prefers-reduced-motion`에만 끄기**: 변경 최소나 일반 사용자에겐 현재 섬광이 그대로 → 오너가 지적한 문제(모든 사용자 대상)를 못 푼다. 기각.
- **CSS/포스트프로세싱 레이어에서 후처리**: 소스가 아니라 말단에서 덧칠 → 세 소비처에 중복 보정, 단일 진실원 위배. 기각.

## 트레이드오프
- (+) 광과민성 위험 큰 전체화면 백색 플래시·반복 미니플래시 제거. 시네마틱 느낌은 글로우로 보존.
- (+) 변경 범위 좁음(타임라인 상수 + 1줄 정규화 + 테스트). 단일 진실원이라 세 소비처 자동 정합.
- (−) 오마주의 "백색 폭발" 임팩트가 약해짐 — 의도된 절충(접근성 우선).
- (−) `burstPeak` 의미가 "히어로 정규화 분모"에서 "글로우 세기"로 바뀜 → HeroPortrait가 `burstPeak`에 더는 의존하지 않음(결합 감소, 긍정적 부작용).

## 영향
- `cerebro-timeline.ts`: `intro`/`loop`/`bloom` 상수, `introState`/`loopState`, `TimelineState` 주석.
- `HeroPortrait.tsx`: `uBurst` 산출 1줄.
- `cerebro-timeline.test.ts`: bgWhiteT 상한·루프 무섬광 회귀 테스트 추가. 기존 flash 상한 테스트는 그대로 통과(상한이 0.28로 낮아짐).
